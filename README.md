# Avito Lead Scoring Solution

Решение тестового задания Авито на предсказание вероятности успешного обращения.

## Описание подхода

**Задача:**  
Предсказать вероятность (score) успешного целевого действия в течение 5 дней после назначения обращения.

**Что использовал:**
- Все готовые табличные признаки из `train.csv` / `test.csv`
- CatBoostClassifier (open-source)
- Time-based валидация (первые 80% дат — обучение, последние 20% — валидация)
- class_weights для борьбы с дисбалансом классов

**Воспроизводимость:**
- random_seed = 42
- Код полностью воспроизводим

**Что можно улучшить:**
- Feature engineering из `events.csv`
- Blending моделей
