# avito-Data-Sciense-Bootcamp-2026_1
Решение тестового задания Авито

## Описание подхода

**Задача:**  
Предсказать вероятность успешного целевого действия в течение 5 дней после назначения обращения.

**Что использовалось:**
1) Все готовые табличные признаки из `train.csv` / `test.csv`
2) CatBoostClassifier (open-source)
3) Time-based валидация (первые 80% дат — обучение, последние 20% — валидация)
4) class_weights для борьбы с дисбалансом классов

### Почему именно CatBoost?

- **CatBoost** лучше всего работает «из коробки» с категориальными признаками.
- Автоматически обрабатывает пропуски.
- Хорошо справляется с несбалансированными данными.
- Показывает высокое качество на табличных задачах с минимальной предобработкой.
- Удобен и стабилен.

**Что пробовал раньше:**
- Logistic Regression (baseline) — быстро, но слабое качество.
- Пробовал LightGBM и XGBoost — они тоже хороши, но CatBoost показал лучший результат с наименьшими усилиями на этом этапе.

## Как запустить

1. Скачать данные по ссылке из задания
2. Положить их в папку `data/`
3. Установить зависимости: `pip install pandas catboost scikit-learn`
4. Запустить `solution.ipynb`
