# avito-Data-Sciense-Bootcamp-2026_1
Решение тестового задания Авито

## Описание подхода

**Задача:**  
Предсказать вероятность успешного целевого действия в течение 5 дней после назначения обращения.

**Что использовалось:**
- Все готовые табличные признаки из `train.csv` / `test.csv`
- CatBoostClassifier (open-source)
- Time-based валидация (первые 80% дат — обучение, последние 20% — валидация)
- class_weights для борьбы с дисбалансом классов

**Воспроизводимость:**
- random_seed = 42
- Код полностью воспроизводим
